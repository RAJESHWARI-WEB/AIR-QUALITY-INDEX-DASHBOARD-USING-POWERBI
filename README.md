# 🌍 Air Quality Index Dashboard using Power BI

## 📌 Project Overview
The Air Quality Index (AQI) Dashboard is an interactive data visualization project developed using Power BI.  
This dashboard helps analyze air pollution levels and provides insights into environmental conditions using various visualizations.

It presents important air quality indicators such as AQI values, pollutant levels, and trends through charts and graphs.

---

## 🎯 Objectives
- Analyze air quality data using visualizations
- Monitor pollution levels through AQI indicators
- Identify trends and patterns in air pollution
- Create an interactive environmental monitoring dashboard

---

## 🛠 Tools & Technologies Used
- Power BI Desktop
- Data Visualization
- Data Analysis
- Data Cleaning

---

## 📊 Dashboard Features
- AQI level visualization
- Pollutant level analysis (PM2.5, PM10, etc.)
- Interactive charts and graphs
- Filters and slicers for better analysis
- Easy-to-understand dashboard design
